Detailed Description

Provides a Behavior(Of T) which forwards the associated object's binding context to the behavior. This is NOT done by default because behaviors can be shared if they are applied using a Style. In that case, there is not a 1:1 relationship. This enforces the 1:1 relationship but disallows the application via a shared resource.
